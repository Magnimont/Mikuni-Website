EpicBot

A simple, multipurpose Discord bot.

Utility

  • Announce This will make an announcement in the desired channel.
  • Giveaway This will start a giveaway in the desired channel.
  • Translate This will translate the text to the desired language.
  • Poll This will start a poll.
  • Countdown This will start a countdown for you.
  • Create Invite This will create a permanent invite link for the server.

Moderation

  • Clear This will delete the amount of messages.
  • Kick This will kick the user from the server.
  • Ban This will ban the user from the server.
  • Unban This will unban the user from the server.
  • Warn This will warn the user.
  • Create Channel This will create a text-channel with the name mentioned.
  • Delete Channel This will delete the text-channel mentioned.
  • Add Role This will add the role mentioned to the user.
  • Remove Role This will remove the role mentioned from the user.

Fun

  • Hack This will hack the user(not real hack).
  • OWO This will convert your text and make it look adorable.
  • Meme will show some memes.

  • Predict The bot will predict the answer to your question.
  • Dice The bot will roll a dice for you.
  • Coin Flip The bot will flip a coin for you.

Bot

  • Prefix It will change the bots prefix for the server.
  • Invite This will send you a link with which you can invite EpicBot to your own server.
  • Discord This will send you a link with which you can join out Discord server.
  • BotInfo It will show the info about the EpicBot.
  • Bug Report It will give you a link with which you can report a bug.

Info

  • Covid-19 Gets covid-19 stats for your country.
  • UserInfo Gets info about the user.
  • ServerInfo Gets info about the server. "Manage Server" permissions will be needed.
  • Avatar Gets the avatar of the user.
